# 🎓 毕业季 · 流光手势画廊

> 一款基于浏览器的手势交互装置，通过摄像头识别手势，控制3D粒子特效与照片画廊。  
> 融合 **Three.js** 粒子系统 + **MediaPipe** 实时手势识别，打造沉浸式毕业季体验。

**[🔗 在线体验](https://graduation-light-gallery.vercel.app)** | [📁 源码仓库](https://github.com/你的用户名/graduation-light-gallery)

---

## ✨ 核心功能

| 手势 | 效果 | 交互说明 |
|------|------|----------|
| ✌️ **剪刀手** | 金色粒子文字“毕业快乐”悬浮空中 | 两指伸出，粒子组成清晰中文 |
| ✊ **握拳** | 回到毕业许愿树 | 清除所有特效，重置场景 |
| 🖐️ **手掌 + 挥手** | 进入时空隧道画廊 | 手掌张开并左右挥动 |
| 🖐️ **手掌停留3秒** | 全屏放大当前照片 | 画廊模式下，手掌静止3秒 |
| 全屏中**挥手** | 切换下一张照片 | 全屏模式下继续用手掌控制 |
| 全屏中**握拳** | 退出全屏 | 返回画廊模式 |

---

## 🛠️ 技术栈

| 技术 | 用途 |
|------|------|
| **Three.js (r160)** | 3D场景渲染、粒子系统、Bloom光效 |
| **MediaPipe Hands** | 实时手势识别（21个手部关键点） |
| **EffectComposer** | 后期特效合成（UnrealBloomPass） |
| **InstancedMesh** | 高性能实例化粒子渲染（7000个粒子） |
| **Canvas 2D** | 中文字符像素采样，生成粒子文字 |
| **getUserMedia** | 摄像头流捕获与镜像处理 |

---

## 🎨 视觉效果

| 模块 | 实现细节 |
|------|----------|
| **毕业许愿树** | 粒子组成圆锥形树冠 + 金色果实点缀 + 飘落花瓣 |
| **时空隧道** | 粒子沿螺旋轨道运动，照片环绕旋转 |
| **金色文字** | 高分辨率像素采样(1200x400)，step=2，粒子半径0.05 |
| **背景特效** | 旋转光环、飘浮光球、星空粒子、Bloom光晕 |
